# Potato Leaf Disease Detection

A deep learning-based image classification system to detect potato leaf diseases using Convolutional Neural Networks (CNNs).

---

## Highlights

* Achieved ~95% accuracy on real-world dataset
* Implemented data augmentation for better generalization
* Designed a modular ML pipeline (data loading, model, prediction)

---

## Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* OpenCV


---

## Workflow

1. Load and preprocess image dataset
2. Apply data augmentation
3. Train CNN model
4. Evaluate model on test data
5. Predict disease class for new images


---

## How to Run

1. Clone the repository
2. Install dependencies

   ```
   pip install -r requirements.txt
   ```
3. Run the training script or notebook
4. Update dataset path as required


---

## Applications

This model can assist farmers in early detection of potato leaf diseases, helping improve crop health and yield.
